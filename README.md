# 08 JavaScript Projects

A collection of JavaScript projects created while learning and experimenting through hands-on practice.

This repository is meant to document my learning journey and serve as a reference for anyone exploring core JavaScript and frontend concepts. No frameworks used, only focused on fundamentals

---

## Projects in this repository:
1. **Amazon Clone** — A dynamic clone of the Amazon shopping website built to practice core JavaScript concepts and UI interactions.
2. **Clock Timer** — A functional digital clock and timer application focused on JavaScript date and time functions.
3. **Drag and Drop** — An interactive drag-and-drop UI built using mouse events and DOM manipulation.
4. **Image Scroller** — A smooth image scrolling feature implemented to understand scrolling logic and UI behavior.
5. **Image Search Engine** — An image search application that fetches and displays results using the Unsplash API.
6. **Music Player** — A simple browser-based music player, built using the HTML Audio element and custom controls..
7. **QR Code Generator** — A utility that generates QR codes from user input using JavaScript and external API.
8. **TTS (Text-to-Speech – Dummy)** — A basic text-to-speech demo with a dummy login page and random password generator using browser speech synthesis.


> **Note:** Each folder represents an individual project created to practice specific concepts. Some projects may be redesigned, refactored, or expanded over time.

---

## Disclaimer

These projects are built for learning purposes. Code quality, structure, and UI may vary across projects as they reflect different stages of learning.

## How to Run a Project

1. Clone the repository

   ```bash
   git clone https://github.com/your-username/js-projects.git
   ```
2. Open any project folder
3. Open `index.html` in your browser (or use Live Server)
